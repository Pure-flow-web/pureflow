# **App Name**: PureFlow

## Core Features:

- Google Sign-In: One-click authentication using Google Sign-In with Firebase Auth.
- Tasks Management: Create, edit, prioritize (Low/Med/High), and set due dates for tasks using a modal and react-datepicker. Includes the ability to complete or delete tasks. Uses Firestore.
- Notes Management: A text area for creating and auto-saving notes every 2 seconds. Notes are listed with last edited time. Uses Firestore.
- Pomodoro Timer: A 25:00 Pomodoro timer with Start/Stop/Reset functionality.
- Theme Toggle: Dark/Light mode toggle.
- Logout: Logout from the application.

## Style Guidelines:

- Primary color: Light green (#A7F3D0) for a fresh and productive feel.
- Background color: Off-white (#F9FAFA) in light mode and dark gray (#1F2937) in dark mode.
- Accent color: Blue (#60A5FA) for interactive elements and highlights.
- Body and headline font: 'Inter' for a clean and modern feel.
- Use lucide-react icons for a consistent and minimalist style.
- Clean, minimalist layout similar to Todoist and Apple Notes. Sidebar navigation and floating action buttons.
- Subtle transitions and animations for interactive elements.
